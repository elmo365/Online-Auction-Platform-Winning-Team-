# Online-Auction-Platform-Winning-Team-
1. extract Auction folder to C:\xampp\htdocs\
2. run xamp
3. start Sql and Apache on xamp
4. on your browser go to localhost/phpmyadmin
5. create a new database called user_accounts
6. import tables on Phpmyadmin  by importing user_accounts.sql file in Auction folder
7. after import insert new roles (admin, seller, buyer)
8. insert new admin (remember username and password)
9. insert new categories
10. on a new tab go to localhost/auction/index.php
11. sign in with the admin account created.
12. you can logout and create a new user
